# /audit

Quick conversion and quality audit for the landing page.

## Usage
```
/audit                  # Full audit
/audit --mobile         # Mobile focus
/audit --copy           # Copy effectiveness
/audit --trust          # Trust signals
```

## What it checks

### 1. Conversion Elements
- Hero CTA visibility
- Form accessibility
- Payment flow smoothness
- Trust signals presence
- Social proof effectiveness

### 2. Copy Performance
- Headline clarity (5-second test)
- Value proposition strength
- Objection handling
- Call-to-action power

### 3. Technical Quality
- Page load speed
- Mobile responsiveness
- WCAG AA compliance
- Broken links

### 4. Trust Factors
- Testimonials visibility
- Guarantee prominence
- Security badges
- Contact information

## Output Format
```
🎯 Conversion Score: 85/100
📝 Copy Score: 90/100
⚡ Performance Score: 92/100
📱 Mobile Score: 88/100

Top 3 Issues:
1. CTA button below fold on mobile
2. Form validation unclear
3. Trust badges missing

Quick Wins:
✓ Move CTA higher
✓ Add progress indicator
✓ Show testimonials earlier
```
