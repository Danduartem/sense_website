## Responsive Breakpoint Guidelines

### Mobile First Approach (375px baseline)

**Priority Elements Above Fold:**
1. Logo + Navigation (minimal)
2. Headline addressing Amanda's pain
3. Promise transformation 
4. Primary CTA prominent button
5. Trust strip (compressed)

**Content Hierarchy:**
- Headlines: Lora font, bold, high contrast
- Body text: Century font, readable size (16px minimum)
- CTAs: Burgundy background, white text, finger-friendly (44px minimum)
- Images: Optimized for mobile bandwidth

### Tablet Adaptations (744px)

**Layout Changes:**
- Two-column layouts where appropriate
- Larger testimonial cards
- Expanded trust strip with more photos
- Side-by-side problem/solution presentations

### Desktop Enhancements (1280px)

**Advanced Features:**
- Multi-column testimonial grids
- Detailed program module cards
- Enhanced visual hierarchy
- Larger imagery and video elements
- More detailed proof elements
